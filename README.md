## Deploy

This repo includes a GitHub Actions workflow that builds the Vite app and deploys the `dist/` output to GitHub Pages on pushes to `main`.

To enable it in GitHub:

1. Go to **Settings → Pages**
2. Under **Build and deployment**, set **Source** to **GitHub Actions**
